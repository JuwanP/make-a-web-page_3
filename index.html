Procedural Thinking

Procedural thinking involves creating perfectly clear instructions for a computer to follow.

Abstract Thinking

Abstract thinking means finding similarity, or as programmers would say, generality amongst different things, to avoid repetition in coding.

Systems Thinking

Systems thinking happens when you break a big problem down into smaller pieces.

Debugging

Debugging is a systematic process of identifying the cause of the error in the program.

Introduction to HTML

What is a Web Page

A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

What is HTML

HTML stands for: HyperText Markup Language and is the main type of document on the web. However, it's not the only type file, that can exist on the web. Other files include: PDFs, Images, videos etc.

HTML Terms

Tag: An HTML tag is always surrounded with angled brackets. Most tags have an opening tag (<p> for example) and a closing tag, (</p>). Some tags do not require a closing tag (like the <br> tag). Element: An HTML element refers to everything within a set of opening and closing tags. Attribute: This is a property of an HTML element.

CSS Basics

What is CSS?

CSS stands for Cascading Style Sheets and is code written to control the "style" of HTML elements. Cascading means that whatever is applied to the top element must apply to the rest following it, until specified later in the code.
